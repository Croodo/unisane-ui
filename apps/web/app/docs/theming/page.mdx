# Theming

Unisane UI is powered by CSS custom properties. Tokens are generated in `@unisane/tokens` and mapped into Tailwind v4 via `@unisane/tailwind-config`.

## Theme Layers

```css
@import "@unisane/tokens/uni-tokens.css";
@import "@unisane/tokens/uni-theme.css";
@import "@unisane/tailwind-config";
```

## Density Scaling

Use `data-density` on the `html` element to control spacing and typography scale.

```html
<html data-density="compact">
```

## Dark Mode

Apply a `.dark` class to switch tokens to their dark palette.

```html
<html class="dark">
```
