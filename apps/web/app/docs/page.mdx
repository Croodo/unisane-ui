# Unisane UI

Unisane UI is a high-density component library inspired by Material 3, tuned for industrial workflows and data-heavy interfaces.

## Installation

```bash
pnpm add @unisane/ui @unisane/tokens
```

## Quick Start

```tsx
import "@unisane/ui/styles.css";
import { Button, TextField } from "@unisane/ui";
```

## Iconography

Unisane UI uses Material Symbols for iconography. Include the font once in your app layout.

```html
<link
  rel="stylesheet"
  href="https://fonts.googleapis.com/css2?family=Material+Symbols+Outlined:opsz,wght,FILL,GRAD@20..48,100..700,0..1,-50..200"
/>
```
